			TURTLE JUMP GAME
 

REQUIREMENTS:-

1. Python 2.7
2. PyGame library

Steps to Run the Game:

a. Run the turtle_jump.py file.
b. Make sure the images are in the same folder as the python file.
c. Type of algorithm for move selection and number of ply moves can be 
   modified in line 376 and 377 of the source code. 
d. use minimax keyword for running minimax algorithm and alpha-beta keyword 
   for running alpha-beta pruning in line 376 and line 377.
